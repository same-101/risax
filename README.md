# risax
A framework for JS

## example
note: keep the title tag blank
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
</head>
<body>
    <script>
        Return (
          '<h1>Hello, Risax!</h1>'
        );
    </script>
    <!---- Risax CDN ---->
    <script src="https://raw.githubusercontent.com/same-101/risax/main/risax.js"></script>
</body>
</html>
```
